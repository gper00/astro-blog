# Belajar Algoritma & Struktur Data dengan Java

Bagi banyak mahasiswa, perjalanan belajar algoritma dan struktur data biasanya dimulai dengan **C atau C++**. Bahasa itu memang sering dijadikan dasar karena dekat dengan konsep memori, pointer, dan performa.

Namun, dalam praktik modern, Java menjadi salah satu bahasa populer untuk mempelajari **algoritma dan struktur data**. Java relatif lebih aman (tidak perlu berurusan langsung dengan pointer mentah), sintaksnya rapi, dan digunakan luas dalam dunia kerja. Karena itu, kita akan melanjutkan perjalanan memahami algoritma dan struktur data dengan Java.

---

## Mengapa Algoritma dan Struktur Data Penting?
Algoritma dan struktur data adalah fondasi dalam ilmu komputer.

- **Algoritma**: langkah-langkah logis untuk menyelesaikan masalah.
- **Struktur data**: cara menyimpan dan mengatur data agar algoritma bisa bekerja efektif.

Tanpa pemahaman keduanya, program hanya akan jadi kumpulan kode yang sulit dioptimalkan dan sulit dikembangkan.
