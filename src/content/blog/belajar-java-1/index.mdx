## Instalasi Java & Setup VSCode

Sebelum ngoding, kita butuh “alat tempur” dulu. Ada dua hal penting: **JDK** dan **VSCode**.

### 1. Install Java JDK

**JDK (Java Development Kit)** adalah paket yang isinya compiler (`javac`), runtime (`java`), dan tools lain biar kamu bisa bikin serta jalanin program Java.

Langkah install:

* Download JDK sesuai OS di [Oracle JDK](https://www.oracle.com/java/technologies/javase-downloads.html) atau pakai [OpenJDK](https://adoptium.net/).
* Pilih sesuai OS: Windows, Linux, macOS.
* Install dengan default setting (kalau pemula, nggak usah ubah path dulu).

**Cek instalasi:**

```bash
java -version
javac -version
```

Kalau muncul versi Java, artinya udah beres ✅.
Kalau error → biasanya masalah di **PATH environment**. Untuk Windows, pastikan folder `bin` JDK udah masuk di `Environment Variables`.

### 2. Install & Setup VSCode

1. Download [Visual Studio Code](https://code.visualstudio.com/).
2. Install extension **Java Extension Pack** (biar ada IntelliSense, debugger, dll).
3. Buka VSCode → bikin file baru `Main.java`.
4. Coba tulis kode sederhana dan run.

> Tips: Kalau di Linux, kadang butuh install plugin tambahan atau set PATH biar VSCode bisa detect JDK.

## Struktur Dasar Program Java

Mari kita bikin program paling klasik di dunia programming: **Hello, World!**

```java
public class Main {
    public static void main(String[] args) {
        System.out.println("Hello, Java!");
    }
}
```

### Penjelasan Baris per Baris

* `public class Main`

  * `class` → keyword untuk bikin kelas (semacam blueprint program).
  * `Main` → nama class, **harus sama persis** dengan nama file (`Main.java`).
  * Kenapa huruf besar? → Konvensi Java: nama class pakai **PascalCase**. Kalau beda huruf, compiler bakal error.

* `{ }` → curly braces, dipakai untuk menandai blok kode. Java *case-sensitive* dan juga *brace-sensitive*, jadi jangan sampai salah taruh kurawal.

* `public static void main(String[] args)`

  * `public` → method bisa diakses dari mana aja.
  * `static` → method bisa dipanggil tanpa harus bikin object.
  * `void` → method ini nggak mengembalikan nilai apapun.
  * `main` → entry point, wajib ada di program Java biar bisa jalan.
  * `String[] args` → parameter untuk ambil argumen dari command line (walaupun sering nggak kepake buat pemula).

* `System.out.println("Hello, Java!");`

  * `System` → class bawaan Java.
  * `out` → object output standar (biasanya layar/console).
  * `println()` → method buat cetak teks + enter. Kalau `print()` aja, nggak pakai enter.

### Catatan Penting untuk Pemula

* **Case-sensitive**: `Main` beda dengan `main`. Kalau salah ketik, error.
* **Komentar**:

  ```java
  // Ini komentar 1 baris
  /* Ini komentar
     lebih dari satu baris */
  ```
* **Indentasi**: bukan wajib, tapi bikin kode lebih rapi dan gampang dibaca.

### Output:

```bash
Hello, Java!
```

## Variabel & Tipe Data

Variabel itu kayak “kotak” buat nyimpen data. Kita kasih nama, lalu isi dengan nilai tertentu.

Contoh:

```java
public class Main {
    public static void main(String[] args) {
        int umur = 27;
        double tinggi = 1.75;
        boolean mahasiswa = true;
        String nama = "Umam";

        System.out.println("Halo, nama saya " + nama + ", umur " + umur + " tahun.");
    }
}
```

**Output:**

```
Halo, nama saya Umam, umur 27 tahun.
```

### Tipe Data Dasar

* `int` → bilangan bulat
* `double` → bilangan desimal
* `boolean` → true/false
* `char` → 1 huruf/karakter
* `String` → kumpulan karakter

## Percabangan & Perulangan

Supaya programmu bisa “ngambil keputusan” atau ngulangin tugas tertentu, kita butuh percabangan & perulangan.

### If-Else

```java
int nilai = 75;

if (nilai >= 80) {
    System.out.println("Lulus dengan nilai bagus!");
} else {
    System.out.println("Lulus, tapi bisa lebih baik lagi.");
}
```

### For Loop

```java
for (int i = 1; i <= 5; i++) {
    System.out.println("Iterasi ke-" + i);
}
```

## OOP (Sedikit Bocoran)

Java itu bahasa yang **Object-Oriented Programming**. Intinya, kita bikin class, lalu dari class itu kita buat object.

Contoh sederhana:

```java
class Mobil {
    String merek;
    int tahun;

    void info() {
        System.out.println("Mobil " + merek + ", tahun " + tahun);
    }
}

public class Main {
    public static void main(String[] args) {
        Mobil m1 = new Mobil();
        m1.merek = "Toyota";
        m1.tahun = 2020;
        m1.info();
    }
}
```

**Output:**

```
Mobil Toyota, tahun 2020
```
