## Mengapa Algoritma dan Struktur Data Penting?
Algoritma dan struktur data adalah fondasi dalam ilmu komputer:

- **Algoritma**: langkah-langkah logis untuk menyelesaikan masalah.
- **Struktur data**: cara menyimpan dan mengatur data agar algoritma bisa bekerja efektif.

Tanpa keduanya, program hanya akan menjadi sekumpulan kode yang tidak efisien dan sulit dikembangkan.

# Instalasi, Konfigurasi, dan Dasar Java

Sebelum masuk ke struktur data, kita perlu memastikan lingkungan sudah siap dan memahami dasar Java.

## Instalasi Java JDK
1. Unduh [Java JDK](https://www.oracle.com/java/technologies/downloads/) sesuai sistem operasi.
2. Lakukan instalasi (proses standar, cukup ikuti wizard).
3. Verifikasi instalasi lewat terminal:

```bash
java -version
````

Jika muncul versi Java, berarti instalasi berhasil.

---

## Setup di VSCode

1. Unduh [Visual Studio Code](https://code.visualstudio.com/).

2. Install **Java Extension Pack** dari marketplace VSCode.

3. Buat folder project → buka di VSCode → buat file `Main.java`.

## Struktur Program Java

Program sederhana dalam Java terlihat seperti ini:

```java
public class Main {
    public static void main(String[] args) {
        System.out.println("Hello, Java!");
    }
}
```

Penjelasan singkat:

- `public class Main` → mendefinisikan class dengan nama `Main`.

- `public static void main(String[] args)` → titik masuk program.

- `System.out.println` → mencetak teks ke layar.


Output:

```
Hello, Java!
```

## Variabel dan Tipe Data

Variabel adalah tempat untuk menyimpan data.

Contoh:

```java
public class Main {
    public static void main(String[] args) {
        int umur = 12;
        String nama = "Umam";
        double ipk = 4.00; // aamiiin

        System.out.println("Nama: " + nama);
        System.out.println("Umur: " + umur);
        System.out.println("IPK: " + ipk);
    }
}
```

Tipe data dasar yang sering dipakai:

- `int` → bilangan bulat

- `double` → bilangan desimal

- `boolean` → true/false

- `char` → satu karakter

- `String` → teks

## Percabangan

Percabangan membuat program bisa mengambil keputusan.

```java
int nilai = 80;

if (nilai >= 75) {
    System.out.println("Lulus");
} else {
    System.out.println("Tidak lulus");
}
```

## Perulangan

Perulangan membuat program mengulang perintah secara otomatis.

```java
for (int i = 1; i <= 5; i++) {
    System.out.println("Perulangan ke-" + i);
}
```

## Dasar OOP

Java adalah bahasa berorientasi objek. Contoh sederhana:

```java
class Mobil {
    String merk;
    int tahun;

    Mobil(String m, int t) {
        merk = m;
        tahun = t;
    }

    void info() {
        System.out.println("Mobil: " + merk + ", Tahun: " + tahun);
    }
}

public class Main {
    public static void main(String[] args) {
        Mobil m1 = new Mobil("Toyota", 2020);
        m1.info();
    }
}
```

Output:

```
Mobil: Toyota, Tahun: 2020
```
